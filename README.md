# Learn Web-Development

## Begin with HTML CSS to learn about basic structure of web pages

- Start with HTML CSS [freecodecamp](https://www.freecodecamp.org/learn/responsive-web-design-v9/)
- Combine with book HTML & CSS: Design and Build Websites by Jon Duckett

## Start Learning Javascript to make your webpage dynamic

- Continue freecodecamp with Javascript certication [freecodecamp](https://www.freecodecamp.org/learn/javascript-v9/)
- “Eloquent JavaScript” by Marijn Haverbeke is book to understand Javascript as a programming language
